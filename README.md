# 🚚 Cargo Delivery Delay Prediction System

A Machine Learning-based web application that predicts whether a cargo shipment is likely to be delayed based on shipment details such as route, weather, vehicle type, package information, and delivery mode.

---

## 📌 Project Overview

The Cargo Delivery Delay Prediction System helps logistics companies estimate delivery delays before shipment. The application uses a trained Machine Learning model and provides the delay probability along with the risk level through an interactive Streamlit dashboard.

---

## ✨ Features

- Predict shipment delay
- Calculate delay probability
- Display Low, Medium, or High Risk
- Interactive and user-friendly Streamlit interface
- Real-time prediction using Machine Learning

```bash

```

Move into the project folder:


Install dependencies:

```bash

```

Run the application:

```bash
streamlit run app.py
```
